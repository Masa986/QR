# 出退室管理システム — セットアップガイド

## 📁 ファイル構成

```
index.html      ← メインアプリ（スマホ用・出退室操作）
monitor.html    ← モニター画面（大画面表示用）
qr.html         ← QRコード表示・印刷用
meibo.csv       ← 名簿データ（編集可）
accesscode.txt  ← アクセスコード（編集可）
```

---

## ⚙️ 使い方

### 1. ファイルの配置
すべてのファイルを **同じフォルダ** に置いてください。
ローカルサーバーか、WebサーバーにアップロードしてURLを発行します。

> ⚠️ `file://` プロトコルでは fetch() が動作しないため、
> ローカルでの確認は簡易HTTPサーバーが必要です。
> 例: `python3 -m http.server 8080`

---

### 2. 名簿の更新（meibo.csv）
以下のフォーマットでCSVを作成・編集してください：

```csv
学年,名前
1年,山田太郎
1年,鈴木花子
2年,田中一郎
2年,佐々木美咲
3年,高橋健太
```

- 1行目はヘッダー（`学年,名前`）
- 学年の種類は自動的に認識されます
- 人数・学年数に制限はありません

---

### 3. アクセスコードの変更（accesscode.txt）
4桁（または任意桁数）の数字を記載するだけです：

```
5556
```

---

### 4. QRコードの設置
`qr.html` をWebブラウザで開くと、`index.html` へのQRコードが表示されます。
印刷して教室入口などに貼り付けてください。

---

## 📊 画面説明

| 画面 | 説明 |
|------|------|
| ロック画面 | アクセスコード入力 |
| 学年選択 | タップして学年を選ぶ |
| 出退室操作 | 入室/退室ボタンで切り替え |
| モニター画面 | 全員の状況を一覧表示 |

---

## 💡 ヒント
- モニター画面は `monitor.html` を直接ブックマークできます
- 出退室の状態はブラウザのlocalStorageに保存されます
- モニター画面は5秒ごとに自動更新、30秒ごとに自動リロードされます
